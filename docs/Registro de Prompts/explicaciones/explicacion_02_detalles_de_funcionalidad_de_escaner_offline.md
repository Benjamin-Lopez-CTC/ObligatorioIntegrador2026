# Detalles de funcionalidad de escáner offline

Para que el escáner funcione offline con alta precisión, reemplazamos el sistema anterior por PaddleOCR, un motor de inteligencia artificial de grado industrial. Como estos motores son muy pesados para ejecutarse en la web, utilizamos ONNX Runtime Web junto con WebAssembly. Esta tecnología permite tomar el "cerebro" comprimido de la IA y ejecutarlo directamente usando el procesador de tu celular a una velocidad casi nativa, sin necesidad de conexión.

Cuando abres la aplicación con internet, el Service Worker descarga silenciosamente este modelo de IA (los archivos .ort) y los guarda permanentemente en la memoria del dispositivo. Luego, al escanear en Modo Avión, tu teléfono le pasa la foto completa a este motor local, el cual es capaz de leer la palabra entera de un solo vistazo y devolverte los números, resolviendo así los problemas de los recortes a ciegas que teníamos antes.

Si el usuario posee una conexión a internet, el sistema utilizará el sistema de reconocimiento anteriormente implementado, Google Gemini 2.5 flash, el cual sigue siendo mejor al momento de reconocer numeros en varias condiciones.

Para poder acceder a todos los datos, una vez se hace login al sistema teniendo una conexión a internet en un dispositivo mobil, el usuario verá una pantalla de carga la cual sirve nada mas que para darle tiempo al sistema de descargar todos los datos a la base de datos local del navegador, la pantalla de carga no es dinamica y siempre va a durar 5.5 segundos, solo se realiza cuando se hace login, si el sistema es visto en un dispositivo de escritorio entonces no se mostrará la pantalla de carga. Una vez terminada la pantalla de carga, el usuario será libre de ir a una zona sin conexión a internet, donde podrán seguir viendo toda la informacion del sitio, y todos los cambios que hagan se guardarán en una cola, la cual una vez restaurada la conexión a internet se sincronizará con el sistema y guardará todos los cambios globalmente.
