# Prompt 25 (Felipe)

Trabajemos sobre la pagina de calculo de costos y ganancias.

En esta pagina tendremos tres tarjetas principales que muestran: inversión total, ganancia bruta y balance neto (profit).

Lista de inversiones registradas, cada una con título, nota y precio. Al pie se muestra el subtotal. Un botón abre un popup para agregar una nueva inversión con esos mismos campos.

Lista de ganancias obtenidas, cada una con título, descripción y monto. Al pie se muestra el subtotal. Un botón abre un popup para registrar una nueva ganancia con esos mismos campos.

Tabla con todos los análisis finalizados. Cada fila muestra: ID, inversión total, ganancia bruta, balance neto, fecha de inicio y fecha de fin. Dos botones por fila:

Detalles: muestra todos los datos de la tabla más el listado completo de inversiones y ganancias registradas en ese análisis.

Eliminar: elimina el registro.

Asegurate de usar los mismos íconos usados en otras tablas.

Un botón en la parte superior de la página permite finalizar el análisis activo. Al hacerlo:

Se guarda automáticamente con la fecha de inicio (cuando fue creado) y la fecha de fin (el momento en que se finaliza).

Se ofrece al usuario la opción de crear un nuevo análisis de inmediato o más tarde.

Si elige crearlo en ese momento, se crea el nuevo análisis.

Si elige hacerlo después, la página muestra un botón/opción visible para iniciar un nuevo análisis cuando lo desee.
