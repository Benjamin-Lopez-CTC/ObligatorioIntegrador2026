# Prompt 31 (B)

El registro de colmenas no permite especificar la cantidad de alzas que tendrá como se solicitó, y con este cambio deberías tambien añadir automaticamente los datos faltantes a todas las colmenas previamente creadas.

Los gráficos de barra del dashboard no pasaron a representar la cantidad de miel extraida, ese grafico debe tomar cada extracción y la fecha y organizar los kilos de miel extraidos cada mes de los ultimos 6 meses de forma dinámica.

Los elementos de la nota técnica no están correctamente alineados, si es necesario para la vista de escritorio ensancha el modal para que quede todo correcto o encuentra otra forma de arreglarlo (ver imagen adjuntada).

También debes añadir datos de prueba de las extracciones y notas tecnicas nuevas para poder corroborar que todo funciona.

Un ultimo cambio, cuando se hace una extracción en masa, cada colmena afectada crea una nota técnica cuya funcion será unicamente notificar que se hizo una cosecha en masa, los otros datos como temperatura y humedad, etc. no se verán afectados.
