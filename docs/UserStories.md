# US1.
Como usuario quiero iniciar sesión en el sistema para acceder al contenido del mismo.

# US2.
Como cliente quiero ingresar colmenas junto a su información para tenerlas registradas en el sistema.

# US3.
Como cliente quiero hacer notas sobre una colmena ya registrada para guardar información que pueda ser útil en un futuro.

# US4.
Como cliente quiero ver los datos de cada colmena por apiario para mantenerme informado.

# US5.
Como cliente quiero realizar un seguimiento del tratamiento de cada colmena para no olvidar ninguna dosis y mantener su salud.

# US6.
Como cliente quiero ver el estado de salud de una colmena para tomar mejores decisiones.

# US7.
Como cliente quiero modificar la información de cada colmena para mantenerla actualizada.

# US8.
Como cliente quiero eliminar una colmena para que en caso de que la misma se pierda o muera, el sistema se mantenga actualizado.

# US9.
Como cliente quiero registrar los movimientos de cada colmena para llevar un historial de los mismos y tener la información disponible en caso de que sea necesaria.

# US10.
Como cliente quiero ver a que apiario pertenece cada colmena para devolverlas a su apiario de origen una vez finalizado el movimiento.

# US11.
Como cliente quiero ver que paso con cada colmena durante todo el año para identificar qué ideas fueron buenas o malas.

# US12.
Como cliente quiero ingresar apiarios junto a su información para tenerlos registrados en el sistema.

# US13.
Como cliente quiero modificar la información de cada apiario para mantenerla actualizada.

# US14.
Como cliente quiero eliminar un apiario para que en caso de que deje de utilizarlo, el sistema se mantenga actualizado.

# US15.
Como cliente quiero asignar cada colmena a su apiario para mantener la organización.

# US16.
Como cliente quiero comparar apiarios y colmenas para tener en cuenta cuales son los mejores.

# US17.
Como cliente quiero ver el estado general de cada apiario para analizar qué acciones influyen en la producción de miel.

# US18.
Como cliente quiero rellenar un formulario con la información solicitada por el ministerio para tenerlo listo y enviarlo en tiempo y forma.

# US19.
Como cliente quiero ver mis ingresos e inversiones para llevar un control de mis ganancias.

# US20.
Como cliente quiero escanear el código numérico en la etiqueta física de cada colmena para ver toda su información en el momento.
