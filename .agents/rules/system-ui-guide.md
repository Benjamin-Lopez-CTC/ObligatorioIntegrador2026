En la carpeta "StitchUI" tendrás todas las vistas del sistema preparadas, tanto en codigo HTML como con imagenes de referencia, para que le implementes el funcionamiento, cada sección mayor estará dividida en carpetas con nombres simples, cuando se te diga "Trabajemos sobre la sección [NOMBRE DE SECCIÓN]" se espera que la UI que utilizes para la funcionalidad del sistema solicitada sea la de esa carpeta.

Ante cambios o vistas extras que se te pida implementar, tendrás una carpeta dentro de StitchUI llamada "DESIGN", la cual tendrá la guia de estilo global del sistema, junto con una imagen de ejemplo.
